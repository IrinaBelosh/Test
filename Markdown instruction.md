## Lists
To create an enumerated list start each line with asterics (*). For example:
* First item
* Second item
* Third item

or start each line with plus (+). For example:
+ First item
+ Second item
+ Third item

To create a numerated list start each line with a number. For example:
1. First item
2. Second item
3. Third item
